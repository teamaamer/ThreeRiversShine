# Three Rivers Shine Cleaning

A modern, professional website for Three Rivers Shine Cleaning - Pittsburgh's premier cleaning service company.

## Features

- **Modern React Application** - Built with React 18 and Vite for optimal performance
- **Smooth Animations** - Framer Motion animations for engaging user experience
- **Responsive Design** - Mobile-first design using Tailwind CSS
- **Multiple Pages**:
  - Home - Hero section, services overview, and CTAs
  - Services - Comprehensive service listings
  - Commercial - Commercial cleaning solutions
  - Residential - Home cleaning services
  - About - Company story and values
  - Contact - Quote request form and contact information

## Tech Stack

- React 18
- Vite
- React Router DOM
- Tailwind CSS
- Framer Motion
- Lucide React Icons

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
